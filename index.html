Featured — job search tool

Job Watcher

A role I liked once had 150+ applicants by the time it showed up in my search — buried past the point any recruiter would still be reading. HR realistically reviews the first 10–20 applicants and moves on. Job Watcher exists to keep me inside that window: it watches LinkedIn continuously, and the moment a matching role goes live, I'm applying in minutes, not days — while I can still get seen.

100+
JDs scanned daily across 8 search terms — typically 3–4 alerts reach my inbox, scored 70+
Under the hood
  • JD Match (50%) — how closely the posting's language matches high-signal phrases from the roles I'm targeting
  • Resume Recency (30%) — how relevant my own experience is to that specific type of role
  • Risk Filter (20%) — catches red flags like contract terms or seniority mismatches before I waste time reading further
  • No LLM, by design — the score is rule-based and fully inspectable; I can see exactly which phrase earned or cost a role its score, and fix the model itself when it's wrong
  • Threshold: 70+ — everything below stays invisible, by design

Grocery Price Bot

Grocery Price Bot

I have Swiggy One. My wife uses Zepto. We kept debating which app to order from — so I built a Telegram bot to settle it. Text it a list exactly like you'd text a person — "curd, bread, paneer, olive oil" — and it replies in under 20 seconds with pincode-specific prices across Instamart, Flipkart Minutes, and Zepto, and where to actually order from.

The real problem wasn't finding prices — every app shows you that. It was that none of them factor in delivery fees, and none of them know I have Swiggy One. So the cheapest-looking app was rarely the cheapest total. This bot solves that specific thing: it computes the actual cost of each order, including delivery economics, and tells you what to do.

Under the hood
  • One API call per item — /v1/groupsearch returns live prices from all 3 platforms in a single call. 5 items = 5 calls, not 15.
  • Three-state availability — ✅ in stock · ❌ not carried · ⚠️ API error. "Not stocked" and "unreachable" are treated differently.
  • No LLM — an earlier version used one to extract prices from search snippets. It hallucinated. Replaced with a structured API.
  • Delivery economics baked in — Swiggy One (free) · Flipkart (free above ₹500) · Zepto (standard fee) — applied before any comparison.
  • Split only if it's worth it — two-app orders recommended only when the saving clears ₹30. Below that, one app wins on simplicity.
